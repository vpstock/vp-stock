Low Stock Email Alerts

Get an automatic email when any item drops to 5 or below.

📋 One-time EmailJS setup (free):
1. Go to emailjs.com → Sign Up (free)
2. Click Add New Service → choose Gmail → connect your Google account
3. Copy your Service ID (e.g. service_xxxxxx)
4. Click Email Templates → Create New Template
5. Set Subject: ⚠️ Low Stock Alert — Victorian Projects
6. Set Body: {{message}} → Save → copy Template ID
7. Go to Account → General → copy Public Key
8. Paste all three below

Cloud Sync Setup

Paste your Firebase Database URL below to sync stock across all devices automatically.

📋 How to get your Firebase URL (2 minutes):
1. Go to console.firebase.google.com
2. Click Create a project → name it "VP Stock"
3. Click Build → Realtime Database → Create database
4. Choose Start in test mode → Enable
5. Copy the URL shown (looks like https://vp-stock-xxxx.firebaseio.com)
6. Paste it above and click Save
